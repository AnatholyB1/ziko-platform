```typescript
const VALID_GRADES = ['a', 'b', 'c', 'd', 'e', 'a-plus'];
const scoredMeals = todayLogs.filter(
  (l) => l.nutriscore_grade && VALID_GRADES.includes(l.nutriscore_grade)
);
```

**3. Compute average grade** (only when scoredMeals.length > 0):
```typescript
const gradeToNum: Record<string, number> = {
  'a-plus': 1, a: 1, b: 2, c: 3, d: 4, e: 5,
};
const numToGrade: Record<number, string> = {
  1: 'a', 2: 'b', 3: 'c', 4: 'd', 5: 'e',
};
const avgNutriscore = scoredMeals.length > 0
  ? numToGrade[
      Math.round(
        scoredMeals.reduce(
          (sum, l) => sum + (gradeToNum[l.nutriscore_grade!] ?? 3),
          0
        ) / scoredMeals.length
      )
    ] ?? 'c'
  : null;
```

Insert daily average Nutri-Score widget** — in the JSX, **after the macros row** (the `View flexDirection: 'row', gap: 10, marginBottom: 16` containing protein/carbs/fat cards — around line 156) and **before the TDEE Calculator link** (the `TouchableOpacity onPress={() => router.push('calculator')}` — around line 158):

```jsx
{/* Daily average Nutri-Score widget */}
{scoredMeals.length > 0 && avgNutriscore && (
  <View style={{

    borderColor: theme.border,
```
