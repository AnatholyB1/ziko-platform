**State:**

Use store: `const { recipes, recipesLoading, recipesError, setRecipes, setRecipesLoading, setRecipesError } = usePantryStore()`

**`fetchSuggestions` async function** (called on button tap and "Régénérer"):
1. `setRecipesLoading(true); setRecipesError(null);`
2. `const { data: { session } } = await supabase.auth.getSession();`
3. `if (!session) { setRecipesError('Non authentifié'); setRecipesLoading(false); return; }`
4. `const res = await fetch(\`\${process.env.EXPO_PUBLIC_API_URL}/pantry/recipes/suggest\`, { method: 'POST', headers: { Authorization: \`Bearer \${session.access_token}\`, 'Content-Type': 'application/json' }, body: JSON.stringify({ preferences: preferences.trim() || undefined }) });`
5. `const data = await res.json();`
6. `if (!res.ok || data.error) { setRecipesError(data.error || t('pantry.recipes_error')); setRecipesLoading(false); return; }`
7. `setRecipes(data.recipes); setRemainingMacros(data.remaining_macros);`
8. `setRecipesLoading(false);`
9. Wrap entire function in try/catch — on catch: `setRecipesError(t('pantry.recipes_error')); setRecipesLoading(false);`

**Skeleton loader** (shown when `recipesLoading` is true):
Render 3 placeholder card views using `Animated` or simple static boxes with opacity 0.4. Use React Native's `Animated.loop` + `Animated.sequence` with opacity 0.3→0.7→0.3 for a pulse effect. Each skeleton card: height 120, borderRadius 12, backgroundColor `theme.colors.surface` (or `#FFFFFF`), marginBottom 12.

**Recipe card component** (inline, for each recipe in `recipes`):

- Row 2: description (14px, `#6B6963`, numberOfLines: 2)
- Row 3: ingredient count (`{recipe.ingredients.length} ingrédients`, 12px, `#6B6963`) + calories chip (`{recipe.macros.calories} kcal`, 12px, primary color `#FF5C1A`, bold)

**Macro budget banner** (shown when `remainingMacros` is not null):

**Error state** (when `recipesError` is not null):

```typescript
const [servings, setServings] = useState(recipe.base_servings);
```
Range: 1–8, integers only. Decrement button disabled when `servings <= 1`, increment button disabled when `servings >= 8`.

**Adjusted macros computation** (per D-08 — client-side only, no backend call):
```typescript
const ratio = servings / recipe.base_servings;
const adjustedMacros = {
  calories: Math.round(recipe.macros.calories * ratio),
  protein_g: Math.round(recipe.macros.protein_g * ratio),
  carbs_g: Math.round(recipe.macros.carbs_g * ratio),
  fat_g: Math.round(recipe.macros.fat_g * ratio),
};
```
This recomputes on every render — whenever `servings` changes, the displayed macros update automatically.
