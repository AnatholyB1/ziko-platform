1. Create `const storageCleanupRouter = new Hono();` — NO authMiddleware on this router.

2. Add `storageCleanupRouter.post('/cron/cleanup', async (c) => { ... })` with this logic:

   a. **Cron auth** (per D-06, same pattern as supplements.ts lines 131-136):
   ```typescript
   const authHeader = c.req.header('authorization');
   const cronSecret = process.env.CRON_SECRET;
   if (cronSecret && authHeader !== `Bearer ${cronSecret}`) {
     return c.json({ error: 'Unauthorized' }, 401);
   }
   ```

   b. **Define retention thresholds** (per D-02):
   ```typescript
   const SCAN_PHOTOS_RETENTION_MS = 90 * 24 * 60 * 60 * 1000; // 90 days
   const EXPORTS_RETENTION_MS = 7 * 24 * 60 * 60 * 1000;       // 7 days
   ```

   c. **Create a reusable `cleanupBucket` async function** that takes `(bucketName: string, retentionMs: number)` and returns `{ deleted: number, errors: string[] }`:
      - Call `supabase.storage.from(bucketName).list('', { limit: 1000 })` to get top-level folders (user ID prefixes). Use the existing `supabase` client already in the file (service key — has cross-user access).
      - If list error or no data, return `{ deleted: 0, errors: ['Failed to list root of {bucketName}: {error.message}'] }`.
      - Compute cutoff: `const cutoff = new Date(Date.now() - retentionMs);`
      - For each folder, use `Promise.allSettled` (per D-05):
        - Call `supabase.storage.from(bucketName).list(folder.name, { limit: 1000 })`
        - Filter files where `new Date(file.created_at) < cutoff`
        - Build full paths: `${folder.name}/${file.name}`
      - Collect all expired paths from all settled promises. For rejected promises, push error string to errors array.
      - If expired paths array is non-empty, call `supabase.storage.from(bucketName).remove(expiredPaths)` (per D-12, D-13 — one batch per bucket).
      - If remove fails, push error to errors array.
      - Return `{ deleted: expiredPaths.length, errors }`.

   d. **Call cleanupBucket** for both buckets inside try/catch:
   ```typescript
   try {
     const scanResult = await cleanupBucket('scan-photos', SCAN_PHOTOS_RETENTION_MS);
     const exportsResult = await cleanupBucket('exports', EXPORTS_RETENTION_MS);
     const allErrors = [...scanResult.errors, ...exportsResult.errors];
     if (allErrors.length > 0) {
       console.warn('[Cron] Cleanup partial errors:', allErrors);
     }
     return c.json({
       success: true,
       deleted: {
         scan_photos: scanResult.deleted,
         exports: exportsResult.deleted,
       },
       errors: allErrors,
     });
   } catch (err: any) {
     console.error('[Cron] Cleanup critical error:', err);
     return c.json({ error: err.message }, 500);
   }
   ```

3. Update the export to include both routers:
   ```typescript
   export { storageRouter, storageCleanupRouter };
   ```
